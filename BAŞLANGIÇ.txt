1️⃣  MongoDB'yi Kurun (2 yöntem)
    
    YÖNTEM A - MongoDB Atlas (ÖNERİLEN - 5 dk):
    ────────────────────────────────────────────
    1. https://www.mongodb.com/cloud/atlas adresine gidin
    2. "Try Free" ile kaydolun
    3. FREE Cluster (M0) oluşturun
    4. Database Access → Add New User → Şifre belirleyin
    5. Network Access → Add IP → 0.0.0.0/0 ekleyin
    6. Clusters → Connect → "Connect your application"
    7. Connection string'i kopyalayın
    
    YÖNTEM B - Yerel MongoDB:
    ────────────────────────────────────────────
    Windows: https://www.mongodb.com/try/download/community
    macOS: brew install mongodb-community
    Linux: sudo apt-get install mongodb

2️⃣  .env.local Dosyası Oluşturun
    
    Proje kök dizininde .env.local dosyası oluşturun:
    
    # MongoDB
    MONGODB_URI=mongodb://localhost:27017/reading-chain
    # MONGODB_URI=mongodb+srv://KULLANICI:ŞİFRE@cluster0.xxxxx.mongodb.net/reading-chain

    # NextAuth (Gerekli!)
    AUTH_SECRET=your-secret-here
    NEXTAUTH_URL=http://localhost:3000

3️⃣  Uygulamayı Başlatın
    
    npm install
    npm run dev
    
    ✨ Tarayıcıda açın: http://localhost:3000

4️⃣  Test Verisi Yükle (Opsiyonel)
    
    npm run seed
    
    Giriş: kullanıcı adı selman, şifre 123456
